# 7.2 할당 연산자

우항에 있는 피연산자의 평과 결과를 좌항에 있는 변수에 할당한다.  
좌항의 변수에 값을 할당하므로 변수 값이 변하는 부수 효과가 있다.

| 할당 연산자 | 예      | 동일 표현  | 부수효과 |
| ----------- | ------- | ---------- | -------- |
| =           | x = 5   | x = 5      | O        |
| +=          | x += 5  | x = x + 5  | O        |
| -=          | x -= 5  | x = x - 5  | O        |
| \*=         | x \*= 5 | x = x \* 5 | O        |
| /=          | x /= 5  | x = x / 5  | O        |
| %=          | x %= 5  | x = x % 5  | O        |

```js
var x;

x = 10;
console.log(x); //10

x += 5; // x = x + 5;
console.log(x); // 15

x -= 5; // x = x - 5;
console.log(x); // 10

x *= 5; //x = x * 5;
console.log(x); // 50

x /= 5; // x = x / 5;
console.log(x); // 10

x %= 5; // x = x % 5
console.log(x); // 5

var str = "My name is ";

//문자열 연결 연산자
str += "Lee"; // str = str + "Lee";
console.log(str); // "My name is Lee"
```

할당문은 값으로 평가되는 표현식인 문으로서 할당된 값으로 평가된다. 따라서 할당문을 다른 변수에 할당할 수 있다. 이러한 특징을 활용해 여러 변수에 동일한 값을 연쇄 할당할 수 있다.

```js
var a, b, c;

// 연쇄 할당. 오른쪽에서 왼쪽으로 진행.
// 1. c = 0 : 0으로 평가된다
// 2. b = 0 : 0으로 평가된다
// 3. a = 0 : 0으로 평가된다
a = b = c = 0;

console.log(a, b, c); // 0 0 0
```
