# 8.1 블록문

0개 이상의 문을 중괄호로 묶은 것으로, 코드 블록 또는 블록을 부른다.  
자바스크립트는 블록문을 하나의 실행 단위로 취급한다. 단독으로 사용할 수도 있으나 일반적으로 제어문이나 함수를 정의할 때 사용한다.

문의 끝에는 세미콜론을 붙이는 게 일반적이나 블록문은 언제나 문의 종료를 의미하는 자체 종결성을 갖는다. 그래서 블록문의 끝에는 세미콜론을 붙이지 않는다.

```js
//블록문
{
  var foo = 10;
}

//제어문
var x = 1;
if (x < 10) {
  x++;
}

//함수 선언문
function sum(a, b) {
  return a + b;
}
```
